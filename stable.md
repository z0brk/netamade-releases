# 哪吒美式 v4.5.0 版本更新日志

> 本次更新重点增强乾坤守护的远程开启控制与监控区域涂抹功能，新增在线音效下载与屏保下载，行程能耗远程通知和远程四路拍照。

## 一、OP 助手

- SR 融合车道控制优化：强制变道时结合 SR 虚实线车道结果决策，提升连续变道成功率
- 降低 SR 融合对变道延迟的影响，减少首次变道的等待确认帧数
- 修复导航控速与自动变道冲突，不再因导航速度控制中断自动变道

## 二、乾坤守护

- **新增远程控制能力**：通过小程序对乾坤守护进行临时设置调整，支持告警配置和远程通知配置，仅本次守护生效，退出后自动恢复
- **新增远程控制能力**：支持通过小程序设置自动结束时间和守护时长（可提前自动关闭守护、远程关闭预警通知、远程调整预警等级）
- **监控区域涂抹**：支持对摄像头画面指定区域进行涂抹，对涂抹区域内的运动变化进行检测（建议只对车身进行涂抹）
- **新增「监控区域强烈运动变化」检测**：检测指定区域内的大幅运动变化并触发预警（配合涂抹使用，不主动涂抹不会启用）
- 支持远程开启乾坤守护（通过哪吒 APP 开启「定时空调」、「座椅通风加热」、「冰箱」后，通过小程序直接开启守护即可，空调自动关闭、关闭座椅通风加热等行为会导致守护直接退出）
- 支持哪吒 L 在临停舒适录制模式下的声音录制问题（不用「露营模式」也可以录制声音了）
- 关闭对车辆的靠近/停留检测以减少误报
- 清理「乾坤视界」的残留行车信息，避免切换后信息残留
- 修复开门晚于解锁时未自动退出的问题
- 修复未开启「下电保持工作流运行」时无法触发乾坤守护退出条件的问题
- 碰撞检测默认值调整为「轻震二级」，降低日常误报概率

## 三、行程能耗

- 新增行程结束时自动发送远程通知，默认关闭，可在设置中选择通知内容章节
- 增加空调运行时长及车内温度记录
- 远程通知增加周期性统计结果推送（本周 / 本月 / 近一周 / 近一月）
  - PS：本周和本月每次推送行程时都会发，若需要周期性报告的选择「近一周」/「近一月」
- 通知内容支持包含地理位置信息（默认关闭），可组合选择续航与状态、行程、能量、驾驶环境等章节

## 四、车机音效

- **新增在线音效下载**：可在音效设置中浏览和下载音效包，下载后自动安装至本地音效库
- 声音库列表按添加时间倒序排列
- 修复替换同名音频文件后 SoundPool 仍播放旧缓存音频的问题
- 锁车、解锁音效响应速度更快（默认未开启，开启后可以和原车的锁车音效一样快，解锁建议设置 500ms 以上的延迟）
- 精简内置音效资源，减小安装包体积（以后都用「在线音效下载」替代）

## 五、系统屏保

- **新增在线屏保下载**：可在屏保设置中浏览和下载屏保资源
- **新增 5 种屏幕目标适配**：L 主屏全屏 / L 主屏半屏 / L 副屏 / S、GT 主屏 / S、GT 副屏，自动适配当前车型
- 支持按媒体类型（图片 / 视频）和适配性筛选搜索
- 精简内置屏保资源，减小安装包体积

## 六、智能场景-锁车提醒

- 支持非 L 车型单独控制是否进行车内钥匙检测(L 不能检测钥匙，所以此功能对 L 无效)
- 使用新的解闭锁判断时机，提升检测准确度
- 修复驻车照片重复使用的问题
- 修复哪吒 L 锁车误报未锁车的提醒问题
- 修复锁车期间系统断开 U 盘导致提醒发送失败的问题
- 兼容 22S「哪吒卫士」运行期间的锁车四路拍照（非「哪吒卫士」运行期间的四路拍照也更稳定了）

## 七、视频库

- 恢复完整倍速播放支持：0.2x / 0.5x / 1x / 2x / 4x / 8x / 16x / 32x
- 限制视角面板自动展开次数，避免频繁自动弹开干扰查看

## 八、远程车控
> 请查看股东群群公告教程后再使用远程车控功能，确保车机娱乐网络畅通

- 新增远程四路拍照功能，拍照后可在小程序直接显示（支持全车型，仅永久股东可用）
- 支持在远程唤醒车机时绕过车机摄像头限制开启摄像头（这就是能够实现远程开启守护的根本）
- 优化实时画面的稳定性，现在获取画面更快，画面更清晰，延迟更低
- 车辆首页「寻车」入口改为「远程画面」,点击直达远程画面 Tab
- 修复空调空气循环模式错误的问题，修复空调温度同步只能关闭无法开启
- 新增车窗控制页，整合天窗与遮阳帘控制，支持四窗独立 0-100% 开度调节与一键透气/全开/全关

## 九、启动器模式切换

- 新增极客与桌面模式的可视化选择对话框，含真实界面预览（更新后会弹出引导设置提示）
- 切换后增加聚光入口引导与位置化语音提醒
- 统一桌面 Dock 的切换桌面入口与图标

## 十、稳定性与优化

- 修复「后排安全带警报」自动关闭不生效的问题，已默认全部开启（此功能可解决系统转向灯无声音的问题）
- 优化 S、GT、猎装车型的车外大喇叭开启速度，与 L 车型一样快了
- 「探路模式」增加独立视角参数
- 盲区悬浮窗增加视角组合参数
- 修复系统定位日志空数据覆盖已缓存定位的问题，驻车位置不再因空数据丢失
- 修复应用更新时长时间卡在安装中的问题
