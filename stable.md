# 哪吒美式 2.6.0 版本更新日志
## 基础功能修复
- 修复哪吒美式无限通知问题（优化后启动阶段不再触发无关通知）
- 修复部分场景下车机音效换挡未被打断导致持续播放的异常

## 乾坤守护 功能升级与修复
### 核心问题修复
- 修复U盘空间不足时，退出乾坤守护界面失败的问题
- 修复U盘空间不足时，自动覆盖策略不生效的问题
- 修复U盘空间不足时，录制视频无法正常播放的兼容性问题
- 修复乾坤守护运行期间后视镜自动折叠的异常逻辑

### 功能体验增强
- 乾坤守护期间新增锁车/解锁提示音反馈
- 重构乾坤守护视频列表页面：采用缩略图展示模式，优化视频浏览效率
- 新增视频批量操作功能：支持批量导出、批量标记重要（标记后不会被自动覆盖）、批量删除
- 优化视频播放页面交互：
    - 支持全屏查看、双击切换全屏、双指缩放、拖动查看局部画面、单击播放/暂停
    - 新增预警时间列表展示、时间轴拖动定位，支持点击事件节点快速跳转播放位置

### 存储管理优化
- 新增联动存储清理机制：当与原车行车记录仪共用U盘时，可用空间低于2GB时自动清理原车循环录制视频，清理记录同步保存至乾坤守护历史日志
- 优化乾坤守护录制性能，提升极端场景下的录制稳定性

### 运行逻辑优化
- 替换露营模式为隐藏式锁车不下电方案：哪吒APP中车辆状态显示为「临停舒适」（可通过该状态判断乾坤守护是否运行）
- 新增APP联动控制：哪吒APP中关闭「临停舒适」后，乾坤守护自动退出录制并保存记录（关闭后可恢复原车空调功能）
- 功能命名优化：
    - 将「锁车哨兵」重命名为「下车开启哪吒卫士」（避免与乾坤守护功能混淆）
    - 将「乾坤守护」智能场景重命名为「下车开启乾坤守护」（触发条件：P档+安全带解开）
- 调整启动电量限制：需满足电量≥15%或处于充电状态方可启动（功能待验证），电量低于10%时自动退出守护

### 备注
- 乾坤守护启动电量限制功能暂为灰度测试版本，实际生效逻辑以车辆硬件适配情况为准

# 哪吒美式 2.5.0 版本更新内容

## 🎉 功能更新速览
- 【数字座舱】全新推出的「数字座舱」功能，包含「实时数据」和「控制中心」两大核心板块，支持悬浮窗“总是显示”、“指定应用显示/隐藏”以及位置锁定功能
- 【状态面板】原仪表盘升级为「状态面板」，支持完全自定义！长按即可进入编辑模式，支持拖拽排序、添加/删除数据源，想看什么你说了算
- 【状态面板】新增大量实用数据源：轮胎压力、轮胎温度、平均油耗、动力电池健康度、蓝牙钥匙电量、已用内存等
- 【顶部控制】顶部控制栏回归并增强，新增重启美式、后台运行、剪贴板、偏好设置、文件互传快速入口，移除播报文字
- 【急刹爆闪】新增急刹车灯爆闪功能，通过检测刹车灯状态实现，响应更灵敏，有效防止追尾
- 【行车音效】新增行车音效功能，音量可随车速动态调整，支持“柴油发动机”和“奥斯汀马丁急速启动引擎”音效
- 【智能场景】新增上下电自动内循环、上电自动关闭 AEB 功能，让用车更省心
- 【智能场景】座椅按摩逻辑优化，副驾开启按摩前会自动检测是否有人乘坐，避免空座按摩
- 【智能场景】座椅加热/通风增加温度阈值控制，温度过高/过低自动关闭，更加智能节能
- 【智能场景】低速安心（自动开启 360）增加 P 档延时关闭功能，挂入 P 档后可延时 0-30 秒再关闭 360，停车更从容

## 🎉 详细功能更新

### 🛸 数字座舱：你的行车管家
全新推出的「数字座舱」功能，集成了「实时数据」和「控制中心」两大核心板块，为您提供更便捷的车辆信息查看和控制体验。
- **实时数据**：
    - **信息丰富**：实时显示车速、转速、功耗、胎压、胎温等关键行车数据。
    - **悬浮展示**：支持以悬浮窗形式常驻屏幕，随时掌握车辆状态。
- **控制中心**：
    - **快捷操作**：集成常用功能快捷开关，一触即达。
    - **智能联动**：可与任务系统联动，实现更复杂的控制逻辑。
- **悬浮窗增强**：
    - **显示模式**：新增“总是显示”、“指定应用显示”、“指定应用隐藏”三种模式，让悬浮窗只在你需要的时候出现。
    - **位置锁定**：支持自由拖动、固定在音乐栏位置或锁定当前位置，防止误触移动。
    - **手动刷新**：允许手动触发顶部应用变化，设置更改后立即生效。

### 📊 状态面板（原仪表盘）：数据一手掌握
- **完全自定义**：长按状态面板即可进入编辑模式，支持对数据源进行拖拽排序、添加和删除操作。
- **丰富数据源**：新增轮胎压力、轮胎温度、平均油耗、动力电池健康度、蓝牙钥匙电量、已用内存等多种数据源，车辆状态一目了然。
- **样式调整**：支持样式调整和实时预览，打造专属于你的个性化面板。

### 👆 顶部控制栏回归
- **功能增强**：顶部控制栏强势回归，并带来更多实用快捷入口。
- **快捷入口**：新增重启美式、后台运行美式、剪贴板、偏好设置、文件互传等快速入口，操作更便捷。
- **界面简洁**：移除了播报文字显示，界面更加清爽干扰更少。

### 🚨 急刹爆闪：安全守护每一程
- **灵敏响应**：不再依赖刹车压力，直接检测刹车灯状态，触发更精准。
- **防追尾**：在急刹车时自动爆闪双闪灯，有效警示后方车辆，提升行车安全。

### 🏎️ 沉浸式行车音效
- **动态音量**：音效音量可随车速动态调整，带来更真实的驾驶体验。
- **多种音效**：内置“柴油发动机”和“奥斯汀马丁急速启动引擎”音效，满足你的赛车梦。
- **车型适配**：哪吒 L (EP32) 支持车外播放，其他车型支持车内播放。

### 🧠 智能场景更懂你
- **自动内循环**：支持上下电自动开启内循环，保持车内空气清新。
- **自动关闭 AEB**：支持上电自动关闭 AEB（自动紧急制动），满足特定驾驶需求。
- **副驾智能按摩**：启动副驾按摩前会自动检测副驾是否有人，无人乘坐时不开启，避免资源浪费。
- **温控更智慧**：座椅加热/通风增加温度阈值控制，当温度达到设定值时自动关闭，舒适又节能。
- **P 档延时关 360**：低速安心功能新增 P 档延时关闭设置，挂入 P 档后 360 画面可延时关闭，让你确认周边环境更安心。

## ⚙️ 其他优化
- **日志过滤**：日志系统支持多标签（Tag）过滤，查找问题更方便。
- **扫码下载**：应用日志点击保存后，支持扫码直接下载日志文件。
- **性能优化**：极大优化了「核心引擎」的启动速度、重连速度以及页面响应速度。
- **界面微调**：优化了空调控制逻辑和 UI 显示，AUTO 和 AC 开关互斥显示，逻辑更清晰。
