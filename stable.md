# 哪吒美式 v4.2.2 版本更新日志

> 本次更新重点补充了任务系统、车辆控制、界面设置和应用更新的可配置能力，并继续修复摄像头管理、亮屏启动等高频使用场景中的问题。

## 任务系统

- 修复探路模式任务参数对话框反复跳转的问题，编辑和保存更稳定
- 探路模式任务的关闭动作合并为统一入口，并新增“自动选择”目标屏幕选项
- 新增“通知弹窗”任务，可在工作流里显示带倒计时的信息提示，弹出后不会阻塞后续任务执行

## 应用投屏

- 投屏悬浮窗支持自由拖动，不再强制限制在屏幕边界内

## 后视镜控制

- 新增“行车中禁止折叠后视镜”保护，检测到行驶中折叠后会自动恢复展开并提醒
- 后视镜折叠保护默认关闭，需要手动开启后才会生效
- 后视镜页面保留“倒车下翻修复”入口，方便在异常时手动执行修复（已知无效果）

## 门锁控制

- 新增“行车中禁止打开后备箱”保护，检测到行驶中后备箱被打开时会自动恢复关闭并提醒

## 行车音效

- 行车音效支持设置自动关闭速度，车速达到阈值后会自动停止播放

## 亮屏启动

- 修复上电时仅设置空气循环模式也会误触发空调开机的问题

## 界面设置

- 将“字体缩放比例”和“显示缩放比例”拆分为两个独立设置，调节时不再互相影响
- 缩放调节支持即时预览，松手后会弹出保存确认，超时未确认会自动恢复
- 优化部分开关在选中状态下的显示效果，状态更清晰

## 应用更新

- 自动检查更新不再依赖 P 挡，非 P 挡时也能按周期后台检查并下载
- 进入更新页面时会识别已下载完成的同版本安装包，直接提供安装入口
- 优化更新下载完成后的安装提示，避免关闭提示时误跳过当前版本

## 乾坤守护

- 新增守护会话摘要提醒，上电后可直接看到尚未查看的重要录制摘要

## 紧急录制

- 触发紧急录制时新增气泡提醒，状态提示更直观

## 存储管理

- U 盘分区现在会标记主驾和副驾位置，选择录制存储设备时更容易区分

## 摄像头管理

- 优化摄像头异常断连后的恢复逻辑，异常中断后更容易自动恢复画面

# 哪吒美式 v4.2.1 版本更新日志

> 本次更新重点修复了语音识别、摄像头管理、上下电联动等问题，并补充了探路模式、智能场景和远程车控的部分能力。

## 语音识别

- 修复语音识别页面崩溃问题，相关设置页可正常进入和编辑

## 任务系统

- 探路模式任务现在支持指定目标屏幕，执行时可直接选择显示到主屏或目标屏幕
- HTTP 任务的局域网端口扫描结果会缓存复用，切换网络或车辆上下电后会自动清空，减少重复扫描等待
- 新增「倒车后视镜下翻修复」任务，可在任务列表中手动执行

注意事项：

- 「倒车后视镜下翻修复」任务目前仅供测试，不保证功能有效

## 智能场景

- 新增「倒车时允许手动调节后视镜」场景，倒车过程中可切换后视镜手动调节模式

## 上下电

- 修复上电时设置「空气循环模式」后误触发空调自动开启的问题，相关行为与设置项含义保持一致

## 摄像头管理

- 修复下电后摄像头画面卡住不更新的问题
- 修复摄像头黑屏后无法恢复的问题，恢复链路更稳定

## 远程车控

- 开启远程车控时增加语音提醒，会播报当前功能的使用限制

## 视频库

- 调整视频缩略图加载节奏，减少 U 盘读取较慢时引发的崩溃

# 哪吒美式 v4.2.0 版本更新日志
> 已知 v4.1.7 版本以来，小部分人存在锁车后摄像头画面卡住不动的问题，因时间问题，会在下个版本会进行修复，当前版本暂未处理

## 高德导航控制（开发中）

新增 HUD 悬浮窗功能，支持将「高德车机版」的导航信息、巡航信息等数据以悬浮窗的形式显示在车机屏幕或 HUD 上：

- 样式「画布」：支持自定义显示的组件、位置、大小等
- 优先显示到 HUD，若 HUD 不可用，则显示到车机屏幕上
- 支持显示的组件：导航信息、巡航信息、车速、转向灯状态等
- 支持多灯红绿灯广播数据解析（需更新美式定制版地图）
- 修复 HUD 运行时状态恢复与过期清理
- 修复车道线、红绿灯、电子眼等多项显示问题

> 目前处于开发阶段，部分功能可能存在问题

## 远程车控

- 支持同步当前车辆的充电数据
- 新增实时画面摄像头隐私控制：可按摄像头单独设置是否允许被远程拉起
- 公网上报车机局域网 IP，支持小程序可编辑下拉框连接
- 新增 PTC 加热和空气净化两个远控命令
- 合并后视镜加热与后挡除霜为统一入口
- 修复空调循环模式开关 ON/OFF 值反转的问题
- 修复续航达成率读取时意外触发模式切换的问题
- 修正空调控制属性映射，与实际功能对齐

## 亮屏启动

- 支持上电设置空调风力和温度
- 支持上电开启空气净化
- 支持上电控制副屏状态（不修改 / 开启 / 关闭）

## 车机音效

- 增加「呱声一片」内置音效
- 修复关闭音效后仍然播放解闭锁音效的问题

## 任务系统

- 新增 HUD 截屏任务参数
- 新增「乾坤视界紧急录制」任务类型，可在工作流中配置触发紧急录像
- 修复 HTTP 任务二次编辑时地址模式回填错误的问题

## 工作流

- 新增触发条件：切换到 D / P / R / N 档位时触发
- 新增触发条件：四个车门（主驾 / 副驾 / 后排左 / 后排右）打开 / 关闭时触发

## 行程能耗

- 新增保存确认语音提醒开关，关闭后仅弹窗提示不播报语音

## 应用投屏

- 支持自定义分辨率保存与选择

## 实时画面

- 优化实时画面的握手逻辑

## 语音识别

- 语音控制应用支持设置别名

## 其他修复

- 修复部分页面 UI 开关状态不会即时更新的问题
