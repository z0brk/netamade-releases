# 哪吒美式 v4.4.4 版本更新日志

> 本次更新重点升级乾坤守护的物理入侵检测（震动/碰撞/拖车哨兵预警），重构远程通知为多策略多渠道体系，智能场景「未锁车提醒」升级为双向「锁车提醒」；OP 助手新增 SR 快照与红灯控速历史；盲区辅助支持方向指示叠加层自定义与更多分辨率选项。

## 一、乾坤守护
> 乾坤守护的风险检测功能迎来史诗级提示✌️

- **新增物理入侵检测**：驻车锁车后通过「量子力学」观察粒子的微观运动实现实时监测震动、碰撞、拖车三类异常事件
- 震动检测：默认开启，支持轻微/中度/严重三档触发等级，可通过灵敏度档位（高/标准/低）整体调节
- 碰撞检测：默认开启，识别碰撞与冲击强度，支持三档触发等级
- 拖车检测：默认开启，识别持续抬车或拖移行为
- 布防延迟：锁车后延迟 5 秒（5~300 秒可配）才开始检测，过滤正常关车门等日常振动
- **震动事件独立提示音**：可为震动事件单独设置预警提示音文件，建议使用股东群群文件的“阿爆-谁拍的.wav”食用达到更佳效果
- **AI 视觉检测优化**：调整距离检测算法，提升靠近/停留/距离过近三种行为触发准确度
- 预警/紧急录制时自动附带车辆四方向（前/后/左/右）驻车照片（会通过「邮箱」等支持图片接收的渠道发送）
- 录制配置设置界面重构，预警事件详情内容优化，展示更清晰的事件触发信息

## 二、智能场景

- **「未锁车提醒」升级为「锁车提醒」**：从仅未锁车提醒扩展为同时覆盖已锁车与未锁车两种方向
- 已锁车通知：检测到车辆成功锁车时发送确认通知，可选附带驻车照片、车辆信息（电量/续航/里程）、最近一次行程能耗
- 未锁车通知：P 档 + 车门关闭 + 车内无人 + 无钥匙时触发，支持未锁车时自动拍照
- 各功能独立开关：锁车照片、未锁车照片、车辆信息推送、行程能耗推送均可单独开关，默认全部开启
- 锁车通知附带最新驻车照片，未锁车通知限制旧照片附件（超过 10 分钟不附带）
- 22S 及智驾包 GT 也支持推送四路的照片
- **修复「隐身休息」执行恢复失败**：踩刹车退出时恢复车内状态、刹车误触发退出、手动执行空操作等问题
- 智能场景设置页全面优化

## 三、远程通知

- **全新的通知策略体系**：从单渠道配置升级为策略→渠道实例两级结构
- 策略：一个"通知方案"，包含名称、总开关、一组渠道实例，支持创建、复制、删除
- 同一策略内可添加多个同类型渠道（如配置两个邮件分别发送至不同收件人）
- **新增飞书应用渠道**：支持 App ID + App Secret 认证，通过飞书 IM API 发送消息与图片附件（最大 10MB/张）
- **企业微信支持图片附件**：正文发送成功后逐张独立发送图片消息（JPG/PNG，单张 ≤ 2MB），需手动开启
- **新增「通知记录」页面**：左侧筛选栏（关键词搜索 + 状态/策略/渠道过滤） + 右侧时间线列表，每条记录可展开查看完整详情与失败原因，支持单条删除与一键清空
- 修复飞书机器人签名算法校验问题

## 四、OP 助手

- **新增 SR 快照功能**：在 SR 页面点击"保存快照"按钮，自动截取当前主屏幕截图并保存 SR 状态数据
- **红灯控速历史**：控速记录中区分展示红灯控速与其他控速，红灯类型卡片使用独立配色，显示倒计时、方向、预计到达时间等详情
- **实现小憩模式与露营模式上下电控制**：乾坤守护、小憩模式、露营模式各自独立设置下电动作与自动整车下电开关
- 红灯控速目标速度按 5km/h 分桶合并，减少红灯闪烁导致的历史刷屏
- 移除了变道历史中的多余历史记录

## 五、盲区辅助

- **新增方向指示叠加层**：每个摄像头方向（左/右/前/后）可独立配置闪烁指示边框
- 配置项：开关（默认全开）、颜色（明黄/暖橙/信号红/翡翠绿/青/天蓝/洋红/纯白 8 色预设）、透明度范围、边缘宽度（8~64dp）
- 渲染效果：摄像头画面边缘绘制彩色边框循环闪烁，直观指示转向方向
- **新增多档分辨率**：360P/480P/540P/576P/720P/900P/1080P 七档可选，每个方向与触发源独立配置
- 修复显示到 HUD 时报错问题

## 六、定位授权

- **新增高德逆地理编码 API 定位获取**：当原车地图被冻结无法获取定位时，自动通过高德 Web 服务 API 将 GPS 坐标转换为可读地址
- 挂 P 档时自动调用 API 刷新地址，确保锁车提醒等场景位置精确
- 需自行配置高德 Web 服务 API Key（支持二维码扫描输入）

## 七、工作流

- **新增情景模式工作流条件**：支持以车辆情景模式作为工作流触发或判断条件
- 覆盖 15 种模式：主页、工作、宝宝、陪伴、动感、K 歌、小憩、洗车、浪漫、赛道、宠物、露营、SPA、离车不下电、观影
- 「情景模式变化」触发器：当指定模式进入或退出时触发工作流
- 「情景模式判断」条件：检查当前是否处于/不处于指定情景模式
- 无感进出模式切换时增加语音提醒

## 八、应用更新

- **新增自动安装开关**：默认关闭，开启后下载完成自动安装，关闭时仅下载安装包需手动安装
- 仅在「自动下载更新」开启时显示，形成级联选项（自动检查→自动下载→自动安装）

## 九、行程能耗

- 重置小计里程时自动结束当前行程记录，确保能耗统计与里程重置同步

## 十、其他修复

- **高危确认弹窗优化**：修复输入正确文字后仍无法确认的问题；新增语音播报 3 次后自动填写全部内容，无需手动逐字输入
- **解锁后主页占位问题**：修复解锁车辆后有时仍停留在美式主页面的问题，正确回落至被劫持目标或系统桌面
- **AVM 退出后主屏幕选择器**：修复 360 环视退出后偶发弹出系统主屏幕选择器的问题
- **远程车控实时画面**：修复部分车型远程车控查看不到「实时画面」的问题

注意事项：

- 远程通知策略从旧版自动迁移，首次启动后请检查默认策略配置是否正确
- 飞书应用渠道需自行在飞书开放平台创建应用并获取 App ID 与 App Secret
- 高德逆地理编码 API 需自行申请 Web 服务 API Key

# 哪吒美式 v4.4.3 版本更新日志

> 本次更新新增「隐身休息」智能场景，提供隐私导向的车内休憩体验；修复 OP 助手导航状态丢失、主页劫持崩溃与回落逻辑、盲区辅助画面延迟及亮屏启动失效等稳定性问题。

## 一、智能场景 — 隐身休息

- **新增「隐身休息」模式**：在 P 档且未踩刹车时可自动触发，或通过详情页「立即执行」按钮手动激活
- 激活后自动配置灯光、屏幕、座椅、车窗等多系统状态 —— 关闭外部/内部/氛围灯、副驾/中控/仪表息屏、主副驾座椅后移或躺倒、折叠外后视镜、车窗通风（下降 20%）
- 支持持续抑制亮屏，检测到屏幕亮起后 500ms 内自动再次息屏
- 退出方式：踩刹车持续 3 秒，或挂出 P 档后自动恢复全部原始状态
- **三种预设配置模板**：「短时停车小憩」「等人轻度舒适」「夜间露营」，支持自定义、新建、另存为、重置预设和删除模板

## 二、OP 助手

- **修复导航数据被无关广播覆盖导致导航状态丢失**：非有效导航 payload 不再覆盖已缓存的导航状态，确保 OP 侧持续收到正确导航信息
- **导航数据改为无条件重发**：缓存最后一帧有效导航 JSON，新设备连接或 IP 变更时自动补发；UDP 发送迁移至有序 Channel 队列，避免并发发送冲突
- 仅发送有实际路径/指引内容的导航 payload（含剩余距离、下一条道路名称等），过滤空壳数据

## 三、主页劫持

- **修复无候选时误回美式主页**：当无上一个应用记录或候选均不可用时，优先回落至已配置的主页劫持目标或原车主页桌面，而非直接跳转美式主页，减少对原车交互的干扰

## 四、盲区辅助 x 乾坤视界 x 探路模式

- **画面显示优化**：移除加载状态与淡入淡出动画，摄像头画面直接显示。画面秒出，不再有启动渐显或初始化转圈
- **兼容性提升**: 支持三者同时使用，解决一直以来的摄像头资源冲突问题，避免出现黑屏、白屏或画面延迟等问题

## 五、上下电

- **修复亮屏启动在预唤醒场景失效**：远程开空调等预唤醒场景下，亮屏启动仅在锁车状态被拦截；解锁后仍可正常触发亮屏启动，修复预唤醒时解锁车辆无法自动亮屏的问题

# 哪吒美式 v4.4.1 版本更新日志

> 本次更新重点增强 OP 助手配置管理能力，新增原车哪吒卫士增强功能，优化乾坤守护、盲区辅助和沉浸驾驶体验。

## 一、OP 助手

- **默认关闭**，需在设置中手动启用后才开始扫描设备、维护连接和发送信息
- 优化自动变道策略判定，提高提前靠右优先级并新增触发距离配置，靠右/出口类文案限制在快速或高速道路触发
- 提前靠右和走推荐车道拆分为快速路和普通公路两套独立设置，可分别调整开关、距离、速度阈值和信号延迟
- 支持读取、本地暂存、差异对比和批量同步 OP 配置，可从 OP 管理界面或二维码扫码访问
- 支持将 SR 感知目标数据通过 UDP 发送到 OP，提升环境感知精度
- 每秒自动补发导航信息，避免 OP 侧丢包丢失导航状态；新设备连接或 IP 变更时也会触发补发
- 优化 SR 数据源卡住重连逻辑，减少感知数据中断
- 清空高德电子眼无效限速值，区分字段缺席与 0/-1 清空，避免 Openpilot 继续用过期限速减速
- 连接后自动生成 OP Web 管理页面二维码，支持扫码或点击二维码通过内置浏览器打开

## 二、哪吒卫士

- 新增原车哪吒卫士增强功能，由三个设置页面组成
- **自动开启**：下车时检测安全带解开且档位为 P 档，自动开启原车哪吒卫士；支持倒计时确认、开启前音频通知提示、突破八小时开启限制
- **视频上传**：开启后自动将哪吒卫士录制的视频上传并推送通知到手机；支持设置远程推送频率
- **行人警示**：行人/物体接近时播放车外提示音和闪烁灯光；支持自定义警告音频和灯光闪烁次数

## 三、乾坤守护

- 自动退出时长新增多档可选：5/10/15/30/45 分钟、1/2/4/8/12/24 小时，覆盖短时驻车到长时间停放场景
- 开门退出和解锁退出两个配置项均支持独立设置

## 四、乾坤视界

- 预览悬浮窗支持跨折叠/展开记忆上次选择的视角，切换模式后不再重置
- 合成画面（四路网格视图）支持点击画面四象限区域，自动跳转到对应通道的单路聚焦视图；可在设置中关闭此交互

## 五、智能场景

- 夏日清凉和冬日温暖场景的温度调节支持 0.5°C 步进，温度控制更精细

## 六、沉浸驾驶

- 全部 22 种内置模拟声浪的合成引擎升级为物理模型管线，包含脉冲序列、共振峰滤波、燃烧噪声和排气共振模拟
- 预览试听时长从 3.5 秒延长至 6 秒

## 七、盲区辅助

- 优化摄像头绑定流程，增加 Surface 有效性和状态追踪，减少超时误判和错误解绑
- 修复探路模式切换屏幕时因 Surface 释放顺序导致的黑屏问题

## 八、车辆控制

- 新增「固定热点 IP 网段」功能，可为车机热点设置静态 CIDR（支持 4 个快速模板或自定义 /16 ~ /30），需通过美式上下电或任务系统开启热点才生效

## 九、任务系统

- 变量查看弹窗中支持单个变量删除，无需清空所有变量

## 十、录制

- 修复四路独立 OSD 写入器在多路并发场景下因竞态条件被异步关闭的问题

## 十一、其他

- 修复更新后启动器模式意外恢复为极客模式的问题，已废弃的「标准」/「极简」模式迁移至「桌面」
- 数字座舱 CPU 温度不再自动定时刷新，仅手动触发，降低后台刷新频率

# 哪吒美式 v4.4.0 版本更新日志
> 本次更新重点新增 OP 助手能力，系统优化录制、回放和存储管理体验，并完善高德导航辅助、语音识别、沉浸驾驶、任务系统等常用功能。

## 必读！必读！必读！ ️ 重要提示：录制模式选择建议

经过对「四路独立」与「四路合并」两种录制模式的长期实测，结合车机 SOC 编码器性能上限，现总结如下：

### ❌ 不推荐「四路独立」模式
- 受限于车机芯片编码能力，该模式下录制 车机 360、甚至整车 **必然出现严重卡顿**；
- 即使调至最低画质，帧率仍无法稳定维持在 24FPS，体验极差。

### ✅ 强烈推荐「四路合并」模式
- 即便开启**极限画质**，仍可稳定保持 24FPS 流畅录制；
- 实际清晰度**显著优于**「四路独立」的 1080P@6Mbps；
- 对车机负载更低、内存占用更少，系统更稳定。

### 🎯 推荐录制画质（按存储设备性能分级）

| 档位 | 适用设备 | 分辨率 | 码率 | 说明 |
|------|----------|--------|------|------|
| **流畅优先** | 闪迪 CZ430 / 其他普通 U 盘 | 1080P | 6Mbps | 保障低速写入稳定性，避免丢帧/漏录，满足基本行车记录需求 |
| **标准清晰** | 海康威视 S560 / 闪迪 CZ880 | 2K | 12Mbps | 日常通勤、低速行驶车牌清晰可辨，清晰度与 4K 几乎无异 |
| **极致画质** | 海康威视 S560 / 闪迪 CZ880 | 4K | 20Mbps | 对画质有极致要求且设备散热良好时使用，录制文件体积超大 |

> 💡 **实测发现**：在「四路合并」模式下，2K@12Mbps 与 4K@20Mbps 的实际观感**几乎无差异**——能看清的细节两者皆可呈现，看不清的部分提升分辨率也无济于事。而普通 U 盘因写入速度限制，强行使用高码率易导致录制中断，1080P@6Mbps 是其可靠上限。

### 📌 核心结论
请勿盲目追求「四路独立」。请根据所用存储设备的实际性能选择对应档位：
- **普通 U 盘用户** → 「四路合并 + 1080P@6Mbps」是稳定记录的底线方案；
- **高速固态 U 盘用户** → 「四路合并 + 2K@12Mbps」是兼顾清晰度、流畅度与系统稳定性的最优解。


## 一、OP 助手
> 仅使用于有 OpenPilot 的用户使用，配合「美式定制高德地图」x「CarrotPilot」x「原车 ADAS 数据」实现“遥遥领先”

- 地图领航能力：可对接高德地图车机版广播，将导航、道路、限速等信息同步给 OpenPilot
- 支持按道路类型、匝道、转弯、收费站、测速电子眼和弯道场景进行目标速度控制，并新增更平滑的减速控制
- 自动变道能力：可根据推荐车道、靠右、匝道和转弯等场景生成变道提醒或指令
- 支持实时渲染原车 ADAS 信息，可显示迷你车道图、路沿、静态物体和动态物体名
- 支持向 OP 发送原车 ADAS 的车道线数据，提升 OP 对当前车道环境的感知

## 二、乾坤视界

- 【乾坤视界】增加视频上传功，默认关闭，可自行选择上传普通视频或紧急视频
- 「四路合成」录制时，悬浮窗也支持独立查看单个视角
- 视频库中播放「四路独立」的录制视频时，也支持展示行车信息啦
- 悬浮窗新增更多控制能力，可调整展开尺寸，并支持固定尺寸，避免误触改变大小
- 迷你悬浮窗可设置为显示实时录制帧率，便于判断录制是否流畅
- 「四路合成」和「四路独立」优化视频帧编码策略，降低 CPU 和内存占用
- 录制获取麦克风失败时自动保持无声录制，不再因音频失败中断录制
- 优化四路录制帧率日志，便于定位录制是否稳定

## 三、视频录制

- 新增画质预设，可按「极致流畅、流畅优先、标准清晰、画质优先、极致画质、自定义」选择录制参数
- 新增固定码率档位，支持从 1 Mbps 到 40 Mbps 的多档选择
- 输出分辨率新增 480P、960P、2K 等选项，并保留 720P、1080P、4K 档位
- 录制帧率可选择 15 / 20 / 24 / 25 / 30 FPS；已移除不适合当前摄像头持续输出的 45 / 60 FPS 档位
- 录制设置中明确区分输出分辨率、输出帧率和视频码率，修改后重新启动录制生效

## 四、视频库

- 四路合成视频支持在合成网格和单视角之间切换，查看某一路画面更方便
- 四路独立视频支持网格、单视角、画中画三种播放模式
- 画中画模式下可拖动其它视角位置，便于按当前关注画面调整布局
- 四路独立视频回放时显示行车信息叠加层，回看时可同步查看录制时的关键车辆状态

## 五、乾坤守护

- 录制模式精简为「持续录制」和「预警录制」，移除使用频率较低的「按需录制」和「不录制」
- 预警录制支持预录缓存和预警后延时录制，减少只录到事件后半段的情况
- 锁车不下电开启失败时，若车辆实际未锁车，不再触发直接下电，避免误下电
- 设置页同步支持新的画质预设、分辨率、帧率和码率配置

## 六、存储管理

- 新增「录制存储新手指引」，根据 U 盘位置和性能等级推荐更合适的录制画质方案
- 存储管理界面重新整理信息层级，风险、容量、设备和录制建议更容易查看
- 存储风险记录支持展示设备不可用、写保护、空间不足、写入失败、写入过慢等问题
- 存储优化支持手动滑块调整容量分配，不再只能使用固定推荐值
- 增加最近录制帧率展示，方便判断当前存储与录制配置是否稳定

## 七、高德导航辅助

- 修复红绿灯尾段倒计时延迟，倒计时变化更贴近实际信号灯状态

## 八、任务系统

- 新增「高德地图控制」任务，可在工作流中执行高德搜索、导航、周边搜索、路线控制等能力
- 驾驶模式任务支持多选循环，可只在用户选择的模式之间切换
- 修复驾驶模式切换失败后的处理逻辑，单个模式不可用时会继续尝试下一个可用模式

## 九、语音识别

- 新增「原车离线语音」开关，开启后会阻断原车语音助手联网，使基础语音识别在离线模式下运行
- 语音规则兼容空字段，减少自定义规则字段缺失时的识别异常

## 十、沉浸驾驶

- 合成声浪新增 22 种跑车风格，并支持在选择界面实时试听
- 粒子颜色自定义扩展为 7 色段，可按车速区间设置更细的颜色变化
- 部分高风险启动场景增加输入确认，启动前会明确提示使用影响和注意事项

## 十一、盲区辅助

- 未开启盲区辅助时不再主动连接摄像头，减少与原车记录仪「环录制」的摄像头冲突
- 优化与乾坤视界录制之间的摄像头资源交接，降低录制和辅助预览互相抢占的概率

## 十二、问题修复

- 修复盲区辅助悬浮窗启动时导致控制中心悬浮窗短暂闪烁的问题
- 修复 BLE 自动重连后的写入问题，连接恢复后小蓝灯状态更容易正确同步
- 优化核心引擎启动阶段的蓝牙加载速度，减少启动时等待蓝牙设备刷新带来的卡顿
- 优化激活界面的输入激活码按钮和授权码展示，手动激活入口更清晰
- 移除极简模式，「标准」模式更名为「桌面」
- 移除无用的「倒车后视镜下翻」修复任务，保留正常后视镜下翻控制能力
