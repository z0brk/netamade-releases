# 哪吒美式 3.4.0 版本更新日志
- 【乾坤视界】新增「随档位启停」功能开关，开启后 P 档自动启停，其他档位保持录制不中断
- 修复部分场景下解锁车辆后部分功能未启动、不工作的问题
- 修复探路模式在锁车后未正常关闭的问题
- 新增上电时自动关闭大喇叭，避免声音意外外放
- 优化解锁车音效逻辑：仅在修改开关时，自动将系统锁车 / 解锁提示音设为仅灯光模式
