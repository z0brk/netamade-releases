# 📝 哪吒美式 2.3.0 更新日志

> 新功能：乾坤圈+乾坤视窗 开放**工作流**

## 🎯 乾坤万象 — 智能交互新范式

### 1️⃣ 乾坤圈（AppSwitch）— 应用/任务智能轮换系统
**乾坤圈**是一套高效的应用与任务轮换启动机制，用户可自定义一个混合列表（应用 + 任务），通过单次触发快速切换执行项，大幅提升驾驶场景下的操作效率。

> 💡 **推荐用法**：将【乾坤圈】任务绑定至方向盘按键，实现“一键轮换导航、音乐、语音助手等”，详见股东群演示视频。

#### ✨ 核心功能
- **混合轮换支持**：列表中可同时包含应用快捷方式与系统任务（如大喇叭、灯光控制等）
- **灵活管理**：
    - 单击删除项目
    - 长按置顶，提升常用项优先级
- **语音播报反馈**：启用后自动播报当前激活的应用或任务名称，无需看屏
- **交互悬浮窗（可选）**：
    - 倒计时悬浮窗（3–60 秒可调），防止误触
    - 支持方向盘滚轮切换项目、左右键确认/取消
    - 可选 **网格布局**（多行多列）或 **圆形图标** 显示，兼顾美观与实用性

#### ▶️ 使用方式
在快捷方式或方向盘按键配置中，添加任务：  
**【系统交互】→【乾坤圈】**

---

### 2️⃣ 乾坤视窗（FreeWin）— 应用窗口行为自动化
**乾坤视窗**可根据预设规则，在应用启动时自动调整其显示模式（全屏、分屏、小窗等），实现“开即适配”，告别手动调整。

#### 🪟 四种窗口模式
| 模式   | 说明 |
|--------|------|
| **默认** | 不干预，保持系统原生行为 |
| **全屏** | 自动最大化，适用于导航、视频类应用（如百度地图） |
| **分屏** | 自动进入分屏模式，适合需并行查看信息的场景 |
| **小窗** | 以自由浮窗形式启动，常用于后台播放类应用（如汽水音乐） |

#### 🔧 管理与配置
- **按应用独立设置**窗口行为
- **开关控制**：可随时启用/禁用某应用的自动适配
- **语音提示**：窗口模式切换成功后自动播报结果
- **持久化存储**：配置保存于本地数据库，重启不失效
- **依赖超级权限**：需系统级权限确保窗口控制稳定可靠

#### 🚀 典型场景示例
- 百度地图 → 启动即全屏
- 汽水音乐 → 启动即小窗浮窗

## 🚗 车辆控制与智能场景优化

- **充电自动关灯逻辑增强**：现可正确识别“加热+充电”复合状态，避免低温环境下因状态误判导致灯光未关闭。
- **座椅加热/通风控制优化**：响应更精准，避免频繁启停。
- **后雾灯爆闪修复**：解决白天开启爆闪时前大灯异常同步闪烁的问题。
- **移除“亮屏启动”中的自动解除静音逻辑**：此前该行为可能导致上车时媒体自动播放或车辆无声音。如有需要，请通过自定义任务手动添加【解除静音】操作。

## 🎮 任务与工作流系统

- **工作流功能向普通用户开放**：基础编辑与执行能力已开放，但高风险操作（如系统级指令）仍受限。
- **新增组合任务支持**：可将多个原子任务串联为一个复合动作，例如：
  > “开启大喇叭 → 播放指定音频 → 关闭大喇叭”
  组合任务可绑定至方向盘、快捷方式或乾坤圈，实现复杂操作一键触发。
- **新增车型专属控制**：S 耀世车型用户现可使用 **展翼门 / 剪刀门** 控制任务（需硬件支持）。
