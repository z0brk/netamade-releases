# 🚀 测试版更新

## 功能更新
- 【远程车控】支持通过实时画面查看车内和前挡前视的画面
- 【摄像头管理】优化乾坤视界非运行状态下的摄像头稳定性，提前预热，长期驻留
- 【摄像头管理】极大优化摄像头开启速度，现在乾坤守护、乾坤视界、盲区辅助、探路模式的摄像头都能实现秒开
- 【摄像头管理】修复模式切换盲区黑屏与盲区/远程实时画面冲突
- 【摄像头管理】运行远程唤醒时访问摄像头画面
- 【智能场景】新增「锁车保持上电」场景，离车时自动开启临停舒适/露营模式
- 【车机音效】预开启大喇叭 + 媒体会话快照模型，缩短锁车音效前置等待
- 【智能场景】「低速安心」新增 ACC/LCC 激活期间禁用低速自动开启的默认开关，默认开启
- 【乾坤视界】新增悬浮窗图标显示开关，支持实时生效，默认开启，关闭后将不再显示悬浮窗图标，但相关功能不受影响
- 【续航达成率】修复刷新后续航模式未恢复
- 【语音识别】兼容“重起”应用指令
- 【远程车控】修复公网 MQTT 多次重连失败后无法恢复的问题
- 【远程车控】远程关闭守护时视为非手动退出
