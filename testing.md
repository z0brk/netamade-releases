# 🚀 测试版更新

## 功能更新
- 【工作流】增加 HTTP 调用任务，支持指定端口后自动扫描设备进行 API 调用
- 【乾坤守护】修复「远程关闭」或「工作流触发原车启动」导致的其他模块功能异常问题
- 【乾坤守护】修复因异常退出导致的其他功能不启动的问题
- 【语音识别】修复部分空调和循环的场景导致误识别, 兼容部分车机应用不说出车机版也能识别
- 【摄像头管理】重构当前摄像头管理机制，优化摄像头重连逻辑
- 增加 U 盘写入时存储设备句柄丢失时的错误记录
- 【氛围灯】增加氛围灯律动功能，不限制音乐软件
- 【语音识别】增加「高德车机版」语音控制功能

## ⚠️ 测试必看

- 出现问题/Bug 必须的提供相应时间段内的日志，否则将无法解决
- 出现服务状态异常，先尝试重启美式
- 此为测试版、抢先体验版，不保证各个功能可用性，可能存在不稳定因素
