# 🚀 测试版更新

## 功能更新
- 【行程能耗】行程能耗在 ACC 主动加速时，不识别为接管
- 【智能场景】修复未锁车提醒在半夜充电时的误报问题
- 智能场景增强充电后重置小计里程触发条件，支持插枪重置和开门时重置
- 【乾坤守护】新增守护时车内静音可选开关
- 【车机音效】新增充电插枪与拔枪音效
- 【数字座舱】数据源增加四车门数据，车门全部关闭时显示绿色，有车门打开时显示红色警告
- 【智能场景】新增充电远程通知，监听充电开始和结束并汇总本次充电统计，通过远程通知发送
- 优化探路模式显示问题
- 【行程能耗】优化百公里能耗平均能耗计算问题
- 【行程能耗】修复冷启动里程污染导致行程里程异常跳变
- 优化深度休眠导致的自启动报错问题
- 优化自动重启功能，重启美式时将更加彻底
- 优化多屏幕检测功能，在副屏关闭状态下不进行显示
- 「无感进出优化」功能，当蓝牙或钥匙持续断开 5 分钟 → 保底开启无感
- 【智能场景】优化哪吒卫士自动延长逻辑，非锁车状态不自动延长
- 【智能场景】雨天安心触发条件改为多选档位
