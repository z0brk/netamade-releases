# 🚀 测试版更新

## 功能更新
- 【车机音效】增加 呱声一片.wav 内置音效
- 【远程车控】支持同步当前车辆的充电数据
- 【远程车控】修复导致续航达成率自动切换为动态/标准的问题
- 【任务系统】新增 HUD 截屏任务参数
- 【高德导航控制】*开发中* 增加新功能，支持将「高德车机版」的导航信息、巡航信息等数据以悬浮窗的形式显示在车机屏幕上
  - 样式「画布」：支持自定义显示的组件、位置、大小等
  - 优先显示到 HUD，若 HUD 不可用，则显示到车机屏幕上
  - 支持显示的组件：导航信息、巡航信息、车速、转向灯状态等
  - 修复 HUD 运行时状态恢复与过期清理
  - 目前处于开发阶段，存在很多问题
- 修复 HUD 悬浮窗车道、红绿灯、电子眼等多项显示问题
- 支持多灯红绿灯广播数据(需更新美式定制版地图)
- 【应用投屏】支持自定义分辨率保存与选择
- 【上下电】【亮屏启动】支持上电设置空调风力和温度及空气净化功能
- 【车机音效】修复关闭后仍然播放解闭锁音效的问题
- 【实时画面】优化实时画面的握手逻辑
- 【远程车控】公网上报车机局域网 IP，支持小程序可编辑下拉框连接
- 【语音识别】语言控制应用支持设置别名
