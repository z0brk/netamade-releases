# 🚀 测试版更新

## 功能更新
- 修复使用天窗、弹射模式属性作为工作流时导致等崩溃问题
- 【车机音效】修复 SoundPool 降级回退状态异常
- 【任务系统】工作流支持播放声音时选择音频组
- 【沉浸驾驶】支持显示到副屏/HUD
- 优化无感进出优化功能的文案展示
- 【核心引擎】兼容哪吒 X 的启动（实际哪吒 X 无可用的支持功能）
- 【智驾小蓝灯】写入失败时尝试重连后再次写入，避免连接假死
- 【乾坤守护】新增开门/解锁自动退出并统一带原因的退出提醒
> 锁车守护满指定时长后，开主驾门或解锁车辆触发自动退出，开关与时长（15 分钟步长）可配置，倒计时随守护状态变化重新计时，预警态往返不重置
