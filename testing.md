# 🚀 测试版更新

## 功能更新
- 【状态面板】设备信息新增内部存储清理弹窗与运行内存清理，低存储自动提醒
  - 点击内部存储弹出内部存储清理对话框进行存储清理
  - 点击运行内存调用情况执行一键内存清理
  - 进入页面时在不足 10GB 自动弹出清理对话框并语音提醒
- 【屏保】修复非 L 车型副屏上下电和应用切换时屏保无法启动
- 【存储管理】修复 U 盘名称过长时卡片布局溢出，导致下方 UI 不可见
- 【高德导航】修复红绿灯状态与黄灯尾段显示
- 【摄像头管理】22S 多路摄像头连接适配
- 【屏保】向屏保资源界面和上传对话框展示当前车型推荐分辨率与支持格式
- 【智驾悬浮窗】移除智驾悬浮窗的拖动移动范围限制
- 【主页劫持】修复主页劫持误将「高德地图美式定制版」劫持导致的进入地图功能异常
- 【车辆控制】-【声音】增加「低速行人警示」功能，低速行驶时主动开启远光灯，通过车外大喇叭播放提示音并闪烁灯光，提醒车外行人注意车辆
