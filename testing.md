# 🚀 测试版更新

## 功能更新
- 【工作流】增加 **蓝牙** 及 **WLAN** 的触发条件及任务
  - 触发及条件：蓝牙设备连接、蓝牙设备断开、蓝牙设备已连接、蓝牙设备已断开
  - 触发及条件：WLAN 连接、WLAN 断开、WLAN 已连接、WLAN 已断开
  - 任务：连接蓝牙设备、断开蓝牙设备、连接 WLAN、断开 WLAN
- 【盲区辅助】支持为开门触发单独设置使用「网格视图」、支持设置转向灯和后方来车的退出速度（默认 25）
- 【智驾提醒】增加 ACC/LCC/NNP 的开启、接管和退出提醒
- SOC 温度纠正为 CPU 温度
- 【语音识别】增加语音识别功能，支持通过哪吒语音打开应用、执行任务或工作流

## ⚠️ 测试必看

- 出现问题/Bug 必须的提供相应时间段内的日志，否则将无法解决
- 出现服务状态异常，先尝试重启美式
- 此为测试版、抢先体验版，不保证各个功能可用性，可能存在不稳定因素
