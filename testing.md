# 🚀 测试版更新

## 功能更新
- 【任务系统】修复魔改状态栏状态获取不正确
- 【任务系统】增加「最佳听音位置」和「音响模式」的控制任务
- 【应用下载】支持已安装应用覆盖安装
- 【智驾提醒】恢复智驾提醒的超大音量
- 【通知推送】企业微信默认改为纯文本模式， Markdown 格式改为可配置项
- 【智能场景】-「未锁车提醒」提醒消息附带最后定位信息
- 【定位】获取系统定位信息失败时，使用最新一次历史定位信息
- 【远程车辆网关】支持局域网和公网远程车控
  - 同一局域网内的设备可通过 WebSocket 进行连接，实现实时的车辆状态同步和远程控制指令的执行。
  - 实时读取车辆状态并执行远程控制指令（锁车/解锁、空调温度与风量、前挡除霜、鸣笛/闪灯寻车、车窗通风、天窗、后备箱、座椅加热/通风/按摩、乾坤守护启停等）。
  - 同时支持公网心跳同步，定时上报车辆状态并接收授权的远程控制请求（**不对外开放，仅自用**）。
- 【乾坤守护】调整启动时的系统电量限制检查规则
  - 临停舒适手动启动：>= 30% 通过，10-30% 警告（黄色图标，提示"可能不成功"），< 10% 拒绝
  - 露营模式保持 >= 30% 硬性要求
  - 智能场景自动触发：所有模式均要求 >= 30%，否则不提示
- 【任务系统】支持批量切换/启用/禁用工作流
- 【工作流】支持左右滑动切换标签页
