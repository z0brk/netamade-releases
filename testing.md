# 🚀 测试版更新
- 【工作流任务】增加双屏屏保同步控制功能, 同时开启、关闭和切换下一个屏保的操作
- 移除 HZ_BDCS2_DOORLOCKCMD 避免误触发解锁音效
  - 移除了对 HZ_BDCS2_DOORLOCKCMD 属性的监听和相关处理逻辑
  - 该属性值在下车时可能导致车把手展开，并错误地触发解锁音效
  - 禁用此逻辑以防止不正确的音效播放
- 【车辆控制】-【系统屏保】修复配置同步、删除残留及定时切换问题
  - 修复主副屏屏保配置同步时覆盖共享库导致数据丢失的问题
  - 新增从库中彻底移除配置的功能，修复删除后仍有残留的问题
  - 解耦主副屏播放模式设置，支持独立配置顺序/随机播放
  - 修复定时切换间隔设置不生效的问题
- 修复Android 12+蓝牙状态判断兼容性并增强错误处理
- 支持 S 副屏的主页劫持和屏保自启动
- 启动自定义屏保前自动禁用系统屏保
- 支持在屏幕关闭和车辆锁定时停止屏保
- 摄像头捕捉稳定性提升
- 将摄像头捕获失败恢复机制从全局改为按摄像头独立处理
- 添加下载安装包自动清理安装失败时保留的安装包
- 添加安装前自动清理旧临时 APK 文件功能
- 修复偶发 apk 解析失败导致无法安装
- 数字座舱检测应用切换时只检测主屏
数字座舱添加当前时间数据源
