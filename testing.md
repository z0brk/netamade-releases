# 🚀 测试版更新

## 功能更新
- 【小蓝灯】支持昼夜主题亮度指令
- 【高德导航】融合定位、增强定位开关
- 【高德导航】增加后台定位权限
- 【智能场景】修复离车场景长期被恢复窗口阻断的问题（比如「下车自动开启哪吒卫士」）
- 更新时检测更新包的 MD5值，防止被 CDN 污染导致安装失败
- 【高德导航】优化语音指令识别覆盖
  - 扩展动态导航、搜索、周边和路况指令的口语化匹配
  - 增加固定目的地别名和常见语音误识别修正
  - 过滤否定意图，避免误触发高风险固定命令
- 【远程车控】补齐空调与后视镜指令语义
  - 修复风速 0 未同步关闭空调总电源的问题
  - 新增后视镜展开与切换远程指令
  - 固定后视镜折叠命令写入折叠态
- 【乾坤视界】优化停止录制和应用切换时的预览性能占用
- 【车机音效】修复解锁预开启大喇叭遗留
