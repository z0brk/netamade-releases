# 🚀 测试版更新

## 功能更新
- 【行程能耗】修综合续航正达成率公式并补齐统计续航掉量展示
- 加回手动设置动力电池型号的功能，避免某些车型自动识别失败导致的无法使用
- 高级功能页面重构，支持分类查看与搜索
- 【数字座舱】修正非 L 的实时数据默认位置
- 【乾坤守护】修复预警事件标记丢失问题
- 【乾坤守护】设置界面 UI 优化、增加“预警点亮屏幕”开关设置
- 修复露营模式悬浮窗异常启动的问题
- 【车辆控制】-【声音】新增大喇叭延长控制并接入外放等待延迟
- 【车辆控制】-【主页劫持】将美式设置为系统主页。按下主页键时，将自动跳转到上一个前台应用，而非原车主页（适用于 L > 1.2 版本退出 360 强制返回原车地图的问题）
- 增加自动检测更新和自动下载的功能
