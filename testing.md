# 🚀 测试版更新

## 功能更新
- 【工作流】增加 HTTP 调用任务，支持指定端口后自动扫描设备进行 API 调用
- 【工作流】优化任务的分类

- 【乾坤守护】修复「远程关闭」或「工作流触发原车启动」导致的其他模块功能异常问题
- 【乾坤守护】修复因异常退出导致的其他功能不启动的问题
- 【摄像头管理】重构当前摄像头管理机制，优化摄像头重连逻辑
- 增加 U 盘写入时存储设备句柄丢失时的错误记录

- 【语音识别】修复部分空调和循环的场景导致误识别, 兼容部分车机应用不说出车机版也能识别
- 【语音识别】支持无网时的离线语音识别（感谢 Yangzhen）
- 【语音识别】增加「高德车机版」语音控制功能
- 【语音识别】增加语音「临时关闭功能」，可在不影响功能开关的情况下，临时关闭美式的语音识别，通过以下语音控制方式触发：
  - 关闭美式导航语音、关闭美式任务语音、关闭美式应用语音
  - 打开美式导航语音、打开美式任务语音、打开美式应用语音
  - “关闭”等价于“禁用”和“停用”
  - “打开”等价于“启用”和“开启”、“恢复”
  
- 增加点击顶部版本号快速进入应用更新界面，长按进入应用日志界面
- 【氛围灯】增加氛围灯律动功能，不限制音乐软件(目前还有些问题)，「音乐律动」无音乐软件激活时可以使用其他模式
- 【氛围灯】在原车的颜色组合风格之外，额外增加更多的预设颜色组

## ⚠️ 测试必看

- 出现问题/Bug 必须的提供相应时间段内的日志，否则将无法解决
- 出现服务状态异常，先尝试重启美式
- 此为测试版、抢先体验版，不保证各个功能可用性，可能存在不稳定因素
