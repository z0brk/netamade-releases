# 🚀 测试版更新

## 乾坤守护
- 增加未锁车提醒、远程关闭、电量不足自动退出时的远程通知功能（需要自动配置好远程通知）。
- 增加离车未锁车、未关车门的检测 跳过临时驻车和露营模式下的检测
- 修复录制过程中崩溃的情况


## 盲区辅助
### 🚀 功能调整
- **四向画面独立配置**：左、右、前、后四个画面现在支持完全独立的参数设置，包括窗口位置、缩放比例及触发逻辑。
- **悬浮窗动态锁定机制**：
    - 新增锁定/解锁切换功能。
    - **解锁状态**：支持自由拖动位置及点击切换分辨率（360P/480P/720P）。
    - **锁定状态**：固定窗口位置防止误触，并在右上角显示快速关闭按钮。
- **单独视图显示支持**：支持仅触发前视或仅触发后视画面的显示模式。
- **手动测试触发**：在设置页面新增测试触发按钮，支持在静态环境下预览各种触发模式下的画面表现。
- **恢复默认设置**：为每种触发模式增加了“恢复默认”功能，一键重置窗口位置及配置。

### 交互与 UI 优化
- **界面重构**：重新设计了窗口配置的布局，增加了更清晰的功能操作指南，提升了易用性。
- **默认参数调整**：
    - 初始默认分辨率统一设定为 **360P**。
    - 默认状态下开启左/右转向触发，关闭前/后视触发。
    - 禁用默认的垂直模式和固定位置。

### 问题处理
- 优化了多窗口并发显示时的逻辑，支持针对不同触发源进行精准的配置读取与应用。

## 智驾提醒 (位于【高级功能】-【乾坤万象】-【智驾提醒】页面内)
- 智驾提醒检测修复：支持「请轻转方向盘」和「请立即接管车辆」时提醒

### 🚀 功能新增
- **自定义场景提醒**：针对“脱手一级警告”、“脱手二级警告”及“辅助驾驶退出”三大核心场景，提供完全独立的个性化配置方案。
- **多模态提醒支持**：
  - **语音播报 (TTS)**：支持自定义提醒文案，实现更具人性化的语音交互（如“请轻触方向盘”等）。
  - **个性化音效**：支持绑定单个音频文件或“音频组”（支持组内音效随机/顺序播放），让警报声不再单调。
- **音量直控与自动恢复**：
  - 突破系统音量限制，将以最大化音量提醒你。
  - **自动复位**：播放结束后，系统会自动精准恢复至原始音频状态。
  - 提供 0-100% 的音量百分比控制滑块，支持更细腻的音量表达。
