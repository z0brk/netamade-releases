# 🚀 测试版更新

## 功能更新
- 【行程能耗】修复无掉油行程误显示油耗
- 【行程能耗】修复锁车后行程未正确结束导致解锁时恢复旧行程
- 【上下电】默认不修改空气循环模式
- 【任务系统】新增重启应用任务
- 【语音识别】支持通过语音重启应用
- 【语音识别】增强导航与固定命令的同音词容错及触发词覆盖
- 【语音识别】任务功能默认启用并预置定位修复与雾灯爆闪规则
- 【主页劫持】修复修改开关或参数后需重启应用才能生效的问题
- 【远程车控】实时画面 P2P 推流的基础流程
- 【数字座舱】时间日期格式移除年份显示
- 【任务系统】定位变量读取增加日志输出
- 【远程车辆网关】-> 改名为【远程车控】
- 移除 HUD 导航增强功能，后续 HUD 功能通过【高德地图美式定制版】提供

## ⚠️ 测试必看

- 出现问题/Bug 必须的提供相应时间段内的日志，否则将无法解决
- 出现服务状态异常，先尝试重启美式
- 此为测试版、抢先体验版，不保证各个功能可用性，可能存在不稳定因素
