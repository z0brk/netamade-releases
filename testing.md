# 🚀 测试版更新
- 【上下电】优化车辆电源与语音助手逻辑，以及冻结应用的恢复功能
- 【车机音效】-【驻车开门警示】支持：P 档模式、零速模式、或二选一模式的三种触发条件
- 【智能场景】-【低速安心】添加转向灯亮起时自动开启360功能
- 【车辆控制】-【灯光】支持后雾灯爆闪无限重复功能，支持二次开启时关闭爆闪
- 【车辆控制】-【按键】添加清除原车自定义方向盘按键绑定功能
- 【车辆控制】-「其他配置」添加基于时间的系统主题自动切换功能，可根据时间段自动切换深色/浅色主题，比系统根据光感应的自动切换更灵活
- 【乾坤守护】优化守护服务退出判断逻辑，当锁车不下电和露营模式任一开启时，服务将持续运行
- 【乾坤守护】修复守护记录会话结束原因覆盖逻辑，确保下电模式自动关闭等非用户主动结束原因可被用户主动结束原因覆盖
- 【乾坤守护】优化摄像头捕获失败重启机制
- 【乾坤守护】优化离屏渲染的日志泛滥问题
- 【乾坤守护】修复锁车/解锁提示音打断音乐播放, 避免播放结束后触发其他应用恢复播放
- 【乾坤守护】完善 Guardian 恢复逻辑, 退出和恢复流程中增加后台应用和系统 DVR 的恢复
- 【乾坤守护】修复守护模式退出逻辑, 避免在应用启动过程中踩刹车导致守护模式意外退出; 修复松开刹车时误触退出乾坤守护模式的问题
- 【乾坤守护】乾坤守护启动时主动关闭全车车窗
- 【乾坤守护】预警时增加灯光闪烁提示（只能近光灯）
- 【乾坤守护】预警时的警告提示音增加独立开关，避免有人找不到如何关闭警告提示音
- 【乾坤守护】支持视频编码格式选择：H264 和 H265，可根据设备性能和存储需求选择合适的编码格式 （建议 S 用户使用 H264 保证兼容性）
- 【任务工作流】快捷方式增加无感进出任务类型、系统功能控制的快捷方式支持跟随系统刷新
- 【任务工作流】添加「驾驶模式」和「能源模式」控制任务，支持循环切换
- 【检测更新】添加熊哥赞助的更新镜像
- 【Debug 功能】增加 TBox 调试功能，可挂断紧急呼救，不用小电瓶下电了

- 出现服务状态异常，先尝试重启美式
- 此为测试版、抢先体验版，不保证各个功能可用性，可能存在不稳定因素
