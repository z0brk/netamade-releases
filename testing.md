# 🚀 测试版更新

## 功能更新
- 【图标配置】安装完应用之后将会提示是否添加白名单
- 【声音管理】实现车机声音监听管理功能及测试页
- 【定位修复】定位修复功能 UI 界面优化，增加自定义修复信噪比的设置项
- 【乾坤守护】修复退出后副屏不保持关闭的问题
- 【车机音效】前置执行锁车音效时的其他音乐软件暂停过程
- 【上下电】新增声音音量控制，支持媒体/通话/语音/导航三种控制模式，可根据需要固定声音大小，避免系统每次自动重置
- 【上下电】优化界面UI 效果，拆分不同功能点，避免都挤在一起
- 优化无麦 K 歌功能
