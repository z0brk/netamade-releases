# 🚀 测试版更新

## 功能更新
- 修复未激活设备反复重启导致无法完成服务器授权
- 【视频库】四路合成支持单视角和网格切换，四路独立支持单视角、网格、画中画（可随意拖动其他三个视角位置）
- 【任务系统】驾驶模式任务支持多选循环
- 【沉浸驾驶】合成声浪支持 22 种跑车风格选择与实时试听
- 【高德导航辅助】支持有搜索结果时，通过语音选择第 X 个目的地，支持语音翻页
- 【蓝牙】优化核心引擎启动时的蓝牙加载速度
- 【OP 助手】对接高德车机版广播，实现对 OP 的领航控速、导航及道路信息同步
- 【OP 助手】增加车道图展示、对已知的静态动态物体展示实际名称
- 部分功能启动/开始时强制要求输入文字进行确认，部分问题提前告知
- 移除无用的「倒车后视镜下翻」修复功能
- 移除极简模式，【标准】改为【桌面】
- 【沉浸驾驶】色段自定义改为支持 7 色段
- 【数字座舱】避免盲区辅助悬浮窗启动时导致控制中心悬浮窗闪烁

#### 录制功能优化
- 增加 480P 和 2Mbps 的录制参数配置
- 【乾坤视界】优化「四路独立」录制时的 CPU 和内存占用
- 【乾坤守护】移除使用频率低的「按需录制」和「不录制」模式，只保留「持续录制」和「预警录制」
- 【视频录制】增加画质预设与固定码率档位
- 【乾坤视界】悬浮窗增加更多控制能力，增加锁定尺寸功能
- 【乾坤视界】悬浮窗在「四路合成」录制时也支持独立查看单个视角
- 【乾坤视界】四路合并时的录制也委托给系统 API，不再由 CPU 经手视频帧

## ⚠️ 测试必看

- 出现服务状态异常，先尝试重启美式
- 此为测试版、抢先体验版，不保证各个功能可用性，可能存在不稳定因素
